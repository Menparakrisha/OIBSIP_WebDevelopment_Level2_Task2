# 🌟 Tribute Page

## 📌 Project Overview

This project is a **Tribute Page** developed using **HTML and CSS** as part of the **Oasis Infobyte (OIBSIP) Web Development Internship – Level 2 Task 2**.

The webpage is dedicated to an inspiring personality and highlights their achievements, contributions, and qualities through an attractive and responsive design. The project demonstrates the use of HTML for content structure and CSS for styling and layout.

---

## 🚀 Features

- Responsive Web Design
- Attractive User Interface
- Biography Section
- Images and Visual Content
- Clean Layout and Typography
- Mobile-Friendly Design
- Smooth User Experience
